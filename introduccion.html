Guía del usuario

Funciones de
las teclas

[ 0 ], [ 1 ], [ 2 ], ... [ 9 ] - Teclas numéricas estándar

[ ← ] - Borrar el último carácter

[ + ] - Adición, [ - ] - Sustracción, [ х ] - Multiplicación, [ ÷ ] - División

[ . ] - Indicar la cantidad en decimales

[ C ] - Reinicializar la calculadora

Instrucciones usando
el teclado

La tecla [Enter] corresponde al signo "igual".

Para borrar el último carácter introducido se usa la tecla [Backspace] (tecla con una flecha).

Para introducir el signo de adición se usa la tecla [ + ] arriba o a la derecha del teclado.

Para introducir el signo de sustracción se usa la tecla [ - ] arriba o a la derecha del teclado.

Para introducir el signo de multiplicación se usa la tecla [ * ] a la derecha del teclado.

Para introducir el signo de división se usa la tecla [ / ] a la derecha del teclado.

Para reinicializar la calculadora se puede usar las teclas [Esc]

De la historia de la aparición
de las calculadoras

El antecesor de la calculadora moderna fue abaco (del latín "abaco" significa "tabla").

Un abaco fue una tabla con unas cavidades donde se movían las marcas

de cálculo (unas piedras o pepitas).Dicen que el primer abaco fue notado en

los pueblos de la Babilonia Antigua hace unos 3 mil años A.C.

En la Grecia Antigua, el abaco apareció aproximadamente en el siglo V A.C.
