Objetivos

Desarrollo de la Calculadora Dinámica:


1. Implementar la interfaz de la calculadora con botones para números y operaciones.

2. Utilizar JavaScript para manejar eventos de clic en los botones de la calculadora.

3. Crear funciones en JavaScript para realizar cálculos y mostrar los resultados en tiempo real.

4. Asegurarse de que la calculadora pueda realizar sumas, restas, multiplicaciones y divisiones.

5. Incorporar animaciones CSS y JavaScript para resaltar las teclas presionadas y mostrar transiciones en los resultados.
